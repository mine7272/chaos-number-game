# 혼돈의 숫자 순서 맞추기 (Number Mix Speed Tap)

## 게임 개요
이 게임은 HTML, CSS, JavaScript를 사용하여 만든 모바일 웹 게임입니다. 선택한 난이도에 따라 1부터 20, 30 또는 40까지의 숫자를 순서대로 빠르게 찾아 탭하는 게임으로, 숫자가 아라비아 숫자(1, 2, 3...), 영어 단어(one, two, three...), 한국어(일, 이, 삼...) 형식으로 무작위로 섞여 표시됩니다.

## 게임 방법
1. 원하는 난이도(쉬움, 보통, 어려움)를 선택합니다.
2. '게임 시작' 버튼을 누르면 게임이 시작됩니다.
3. 화면에 무작위로 배치된 숫자들 중 현재 찾아야 할 숫자(처음에는 1)를 찾아 탭합니다.
4. 올바른 숫자를 탭하면 해당 숫자는 비활성화되고, 다음 숫자(2)를 찾아야 합니다.
5. 잘못된 숫자를 탭하면 시각적 피드백과 함께 오류 횟수가 증가합니다.
6. 선택한 난이도에 따라 모든 숫자를 순서대로 찾으면 게임이 종료되고 걸린 시간과 틀린 횟수가 표시됩니다.
7. '다시 하기' 버튼을 눌러 게임을 재시작할 수 있습니다.

## 난이도
- **쉬움**: 1부터 20까지의 숫자
- **보통**: 1부터 30까지의 숫자 (기본 설정)
- **어려움**: 1부터 40까지의 숫자

## 특징
- 숫자가 아라비아 숫자, 영어 단어, 한국어 형식으로 무작위로 표시됩니다.
- 올바른/잘못된 선택에 따른 시각적 및 청각적 피드백을 제공합니다.
- 모바일 환경에 최적화된 반응형 디자인을 적용했습니다.
- 화면 크기가 변경되면 숫자들의 위치가 자동으로 재조정됩니다.
- 게임 완료 시 걸린 시간과 틀린 횟수를 확인할 수 있습니다.

## 실행 방법
1. 브라우저에서 index.html 파일을 열면 게임을 바로 플레이할 수 있습니다.
2. 모바일 기기에서 최적의 경험을 얻을 수 있습니다.

## 개발 정보
- HTML, CSS, JavaScript만을 사용하여 개발되었습니다.
- 외부 라이브러리나 프레임워크 없이 순수 웹 기술만으로 구현되었습니다.
- 모바일 터치 이벤트에 최적화되어 있습니다.
